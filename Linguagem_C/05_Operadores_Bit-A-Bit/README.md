# Operadores bit-a-bit

| & | E | x=a&b |
| --- | --- | --- |
| | | OU | x=a|b |
| ^ | OU EXCLUSIVO | x=a^b |
| ~ | NEGAÇÃO | x=~a |

# Deslocamento de bits

| << | Deslocamento a esquerda | a << b |
| --- | --- | --- |
| >> | Deslocamento a direita | a >> b |
